# Formato de resposta

Toda análise deve seguir:

1. Resumo executivo
2. Leitura da conta
3. Avaliação vs targets
4. Mapa de intenção
5. Diagnóstico de restrições de escala
6. Principais desperdícios
7. Oportunidades de escala
8. Novas keywords sugeridas
9. Termos para negativar
10. Termos para isolar
11. Keywords exatas sugeridas
12. Sugestões de copy/RSA
13. Ações por campanha e grupo
14. Ações prioritárias para amanhã
15. Riscos e confiança

## 1. Resumo executivo

Responder diretamente:
- o que está acontecendo;
- onde está o maior desperdício;
- onde está a maior oportunidade;
- o que fazer amanhã.

## 2. Leitura da conta

Separar:
- branded;
- non-branded;
- Shopping;
- PMax;
- Demand Gen/YouTube/Display quando relevante.

## 3. Avaliação vs targets

Sempre comparar performance com o target correto.

### Branded / Institucional

Targets:
- CPA alvo: < R$10
- ROAS alvo: > 10x
- CPC alvo: < R$0,60

Para cada achado, mostrar:
- CPA atual vs target;
- ROAS atual vs target;
- CPC atual vs target;
- status: dentro do target / atenção / crítico.

### Non-Branded

Targets:
- CPA alvo: < R$20
- ROAS alvo: > 3.5x
- CPC alvo: < R$0,60

Para cada achado, mostrar:
- CPA atual vs target;
- ROAS atual vs target;
- CPC atual vs target;
- status: dentro do target / atenção / crítico.

## 4. Mapa de intenção

Classificar termos/queries em:
- comprador;
- pesquisador;
- comparador;
- concorrente;
- caçador de brinde/cupom;
- ambígua.

## 5. Diagnóstico de restrições de escala

Quando houver `campaign_constraints_analysis.csv`, classificar:
- budget limitado;
- rank limitado;
- budget + rank limitado;
- baixa cobertura;
- baixa presença no topo;
- sem gargalo claro.

Nunca dizer “a conta captura tudo” apenas porque budget lost é 0%.

## 6. Principais desperdícios

Para cada desperdício:
- termo/campanha;
- gasto;
- conversões;
- CPA/ROAS;
- target aplicável;
- motivo;
- ação sugerida;
- risco.

## 7. Oportunidades de escala

Para cada oportunidade:
- termo/campanha;
- CPA/ROAS;
- target aplicável;
- budget lost;
- rank lost;
- Search Console, se houver;
- recomendação;
- condição para escalar.

## 8. Novas keywords sugeridas

Para cada keyword:
- keyword;
- match type;
- intenção;
- grupo sugerido;
- justificativa;
- impressões orgânicas;
- posição orgânica;
- prioridade;
- risco.

## 9. Termos para negativar

Para cada termo:
- termo;
- campanha/grupo;
- gasto;
- conversões;
- intenção;
- justificativa;
- tipo de negativa sugerida;
- risco de falso positivo.

## 10. Termos para isolar

Para cada termo:
- termo;
- por que isolar;
- grupo/campanha sugerido;
- intenção;
- target;
- prioridade.

## 11. Keywords exatas sugeridas

Para cada keyword:
- keyword exata;
- origem: Google Ads ou Search Console;
- intenção;
- métrica de suporte;
- grupo sugerido;
- copy sugerida.

## 12. Sugestões de copy/RSA

Separar por:
- branded;
- non-branded comprador;
- comparador;
- pesquisador;
- produto prioritário.

## 13. Ações por campanha e grupo

Quando houver breakdown granular:
- onde manter;
- onde revisar;
- onde reduzir;
- onde isolar;
- se a ação é global ou específica de campanha/grupo.

## 14. Ações prioritárias para amanhã

Trazer no máximo 5 ações.

Para cada uma:
- ação;
- campanha/grupo/termo;
- métrica usada;
- target comparado;
- impacto esperado;
- risco;
- confiança;
- precisa aprovação humana? sim/não.

## 15. Riscos e confiança

Toda recomendação de alto risco deve trazer:
- justificativa;
- métrica usada;
- escopo sugerido;
- risco de falso positivo;
- necessidade de aprovação humana.

## Regra de estilo

A resposta deve ser:
- direta;
- acionável;
- baseada em dados;
- sem floreio;
- com opinião clara quando os dados forem suficientes.

## Regra crítica

Nunca recomendar:
- aumento de orçamento;
- alteração de tROAS;
- pausa;
- negativa;
- criação de keyword;

sem dizer:
- por quê;
- onde;
- com qual métrica;
- contra qual target;
- qual risco.
